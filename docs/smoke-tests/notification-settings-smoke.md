Smoke Test Checklist - Notification Settings
-----------------------------------------
1. Öffnen der Benachrichtigungseinstellungen: Dialog geöffnet, Form geladen (bestehende Settings geladen).
2. Toggle 'Benachrichtigungen aktivieren' -> deaktiviert alle toggles inkl. Erinnerungen und Sound.
3. Custom Reminder: 90 Minuten hinzugefügt (Einheit Stunden), Badge sichtbar, beim erneuten Hinzufügen verhindert (Duplikate).
4. Quiet Hours aktiviert 22:00-06:00, Statusanzeige entsprechend 'aktiv'.
5. 'Testbenachrichtigung senden' löst Browser-Permission-Prompt aus (falls default) und speichert Notification.
6. Speichern -> Settings bleiben persistent, Badge 'last updated' aktualisiert, Dialog schließt.
